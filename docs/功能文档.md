# 短剧分发系统功能文档

## 1. 项目目标

系统用于把云盘中的短剧自动整理入库，并通过桌面端完成本机下载、视频处理和媒体平台分发。后台负责管理短剧、媒体号、配置、任务策略和运行监控；桌面端负责处理必须发生在本机环境中的动作，例如打开 Chrome、扫码登录、保存平台登录态、转码和上传。

## 2. 角色

### 管理员

- 登录后台管理界面。
- 管理后台用户。
- 配置百度云 App ID、App Key、刷新令牌、检测目录、下载目录策略。
- 管理短剧类别。
- 查看百度云扫描结果、短剧库、集数、封面、简介和入库状态。
- 查看媒体号绑定状态、所属用户、平台登录状态和可用性。
- 配置分发策略并监控任务。

### 桌面端用户

- 使用用户名密码登录桌面端。
- 绑定媒体号，当前优先支持视频号。
- 通过桌面端自动打开 Chrome，扫码登录视频号。
- 让桌面端采集并回传浏览器登录状态。
- 为每个媒体号选择短剧类别和分发节奏。
- 启动或暂停本机分发任务。

## 3. 核心业务流程

### 3.1 百度云短剧入库

1. 管理员在后台配置百度云应用信息、令牌和需要扫描的目录。
2. 后端定时扫描指定年份目录下最新的中文日期目录，例如 `/drama/真人剧/2026/6月13日`。
3. 系统识别短剧目录、剧集视频、封面、简介等素材，并按标题和简介自动分类。
4. 后端把短剧写入库，记录云盘路径、文件 ID、集数、封面、简介、状态和同步时间。
5. 后台展示短剧库，管理员可以手动修正分类、上下架、封面和简介。

### 3.2 媒体号绑定

1. 桌面端用户登录。
2. 用户选择媒体平台标签，当前支持视频号，抖音和 TikTok 显示为预留入口。
3. 桌面端打开独立 Chrome 用户目录，进入平台登录页。
4. 用户扫码登录。
5. 桌面端检测登录完成后导出必要 cookie、local storage 或浏览器 profile 引用。
6. 桌面端把登录态摘要、可用性和设备信息回传后端。
7. 后端记录媒体号、平台、绑定用户、登录态版本、最后验证时间和状态。

### 3.3 分发策略

每个媒体号可以配置：

- 目标短剧类别。
- 每天最多分发短剧数量。
- 两次处理的最小间隔。
- 是否启用。
- 本机工作目录、转码预设、上传前校验规则。

### 3.4 分发任务

1. 桌面端向后端拉取可执行任务。
2. 后端根据媒体号策略、短剧状态、类别和锁定状态创建任务。
3. 桌面端获取短剧下载计划，包含剧名、简介、分集和百度云真实下载地址。
4. 桌面端下载剧集素材。
5. 桌面端调用 FFmpeg 转码或规范化。
6. 桌面端使用 Chrome 自动化上传到视频号。
7. 桌面端把上传结果、截图、平台返回信息和失败原因回传。
8. 后端更新分发任务状态并在后台展示。

## 4. 后台管理功能

### 4.1 登录

- 使用账号密码登录。
- 后端签发 JWT。
- 前端自动携带 Token。
- Token 过期后跳转登录页。

### 4.2 用户管理

- 用户列表。
- 创建、启用、禁用用户。
- 重置密码。
- 记录最后登录时间。

### 4.3 短剧类别管理

- 类别增删改查。
- 类别启用/禁用。
- 类别排序。
- 桌面端读取启用类别。

### 4.4 系统配置

- 百度云应用参数。
- 百度云扫描目录。
- 桌面端版本策略。
- 默认任务节奏。

敏感配置后端加密保存，接口不返回明文密钥。

### 4.5 短剧管理

- 短剧列表、详情、集数列表。
- 封面、简介、类别、状态编辑。
- 触发手动扫描。
- 查看云盘路径、文件大小、下载状态。

### 4.6 媒体号管理

- 媒体号列表。
- 平台：视频号、抖音、TikTok。
- 绑定用户。
- 登录态状态。
- 分发策略。
- 最近任务。

### 4.7 任务监控

- 任务队列。
- 状态统计：待处理、下载中、处理中、上传中、成功、失败、取消。
- 失败原因。
- 桌面端心跳和设备信息。

## 5. 桌面端功能

### 5.1 登录

- 用户名密码登录。
- 本地安全保存访问令牌。
- 支持退出登录和清理本地会话。

### 5.2 媒体号绑定

- 平台 Tab：视频号、抖音、TikTok。
- 视频号支持打开 Chrome 扫码绑定。
- 独立平台 profile，避免污染用户主浏览器。
- 登录态可验证、可刷新、可回传。

### 5.3 任务执行

- 拉取任务。
- 下载素材。
- FFmpeg 转码。
- 平台上传。
- 回传进度和结果。
- 支持暂停、恢复、失败重试。

### 5.4 跨平台

- Windows 和 macOS。
- 自动探测 Chrome、FFmpeg、soffice。
- 支持用户手动指定工具路径。
- 本地数据目录遵循系统规范。

## 6. 已实现范围

- 后端、前端、桌面端完整工程结构。
- 后台登录、用户、类别、配置、短剧、媒体号、任务 API。
- 百度云真实扫描：默认配置导入数据库，扫描 `/drama/真人剧/2026` 最新日期目录，同步剧名、封面、简介、集数和分集路径。
- 自动分类：根据剧名和简介匹配通用、都市、情感、逆袭、神医、古装、美食、科幻、悬疑等类别。
- 下载计划：桌面端按短剧 ID 获取百度云真实下载地址并下载到本机。
- 桌面端视频号绑定：自动打开 Chrome，等待用户扫码，保存独立 profile 登录态，并把媒体号回传后端。
- 桌面端媒体号策略：可拉取类别、查看媒体号、为单个媒体号配置多选类别、每日上限、处理间隔和启用状态。
- 抖音和 TikTok 已保留平台枚举、Tab/适配器扩展边界，后续只需要补具体平台自动化。
